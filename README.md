# Liga de superheroes informáticos 🦸‍♂️🦸‍♀️🦹‍♀️🦹‍♂️

## 📌 Descripción del proyecto
Se deben incorporar distintas funcionalidades:
- Utilizar API para cargar a los supervillanos.
- Emplear "Observable" par obtener los héroes y Promise para eliminar.
- Actualizar los datos utilizando "signal" y "computed".
- Agregar una gráfica de barras con los atributos de todos los héroes (excepto de los villanos):
    - Inteligencia.
    - Fuerza.
    - Velocidad.
    - Durabilidad.
    - Poder.
    - Combate.

##  ⚙ Ejecutar el proyecto

Clonar el repositorio

```bash
git clone https://github.com/Vero7nicaRS/liga-superheroes-informaticos-api-barras-heroes.git
```

Instalar las dependencias:

```bash
npm install
```
Ejecutar el proyecto:
Introducir la siguiente línea de comando: 

```bash
ng serve
```

Abrir el proyecto en el navegador:
```text
http://localhost:4200
```
